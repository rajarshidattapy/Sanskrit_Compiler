# Sanskrit Compiler

A web-based code interpreter that lets you write programs using Hindi/Sanskrit keywords.

## Usage

Write code using these keywords:
- `chhapo` - print output
- `yadi` - if condition  
- `jab tak` - while loop
- `adala badli` - swap variables

## Example

```
a = 5
b = 10
adala badli (a, b)
chhapo a
chhapo b
```

## Setup

1. Clone the repo
2. Run `npm install`
3. Add your Gemini API key to `.env`
4. Run `npm run dev`

That's it.
